Programming: Remove word permutations from list

Permutations are the variations in which a set of items can be arranged
e.g. Given the following set of elements: {a,b,c}, its permutations are {b,a,c}, {c,a,b}, ..., etc.

Considering a word as a set of characters, and given a list of words, keep only the first ocurrence of the set and remove any other permutations present.
e.g. Given the list of words: ['trap', 'crad', 'rapt', 'ted', 'det'], the purged list is ['trap', 'crad', 'ted'] 


          data = ['text', 'tetx', 'extt', 'trex']
          output = []
          maps = []

          for word in data:
              word_map = {}
              for character in word:
                  if not character in word_map.keys():
                      word_map[character] = 1
                  else:
                      word_map[character] = word_map[character] + 1
              if not word_map in maps:
                  maps.append(word_map)
                  output.append(word)

          print(output)
          # ['text', 'trex']
